# Zentiva Brand Guidelines for Presentations

This reference defines the visual standards for creating PowerPoint presentations branded for Zentiva.

## Color Palette

| Element | Hex Code | Usage |
| :--- | :--- | :--- |
| **Zentiva Dark Blue** | `#0C1B7D` | Primary headings, slide headers, dark backgrounds. |
| **Zentiva Teal** | `#00A98F` | Accents, gradients, secondary buttons. |

## Layouts

The template `Brand_v001.pptx` contains specific layouts. Prefer these by name:
- **Title Slide**: For the main deck title.
- **2_Title and Content**: Standard content slide with a title and bullet points.
- **20_Title and Content**: Two-column body layout.
- **10_Title and Content**: Layout including a picture placeholder.
